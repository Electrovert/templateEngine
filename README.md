# Template Engine

## Summary
Javascript documents create an HTML file, and converts it to a Printable PDF using Node.js


## The PDF will be populated with the following:

* Profile image

* User name

### To Execute File:
Open in browser

## Features:
* Two JS Pages
Index.js
generateHTML.js
* Two JSON packages
* Node Modules Folder
* One HTML Page
* One PDF Page
